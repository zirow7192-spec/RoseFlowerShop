# 🌹 Rose Flower Shop - Οδηγίες Χρήσης Admin

## 📱 Πώς να Χρησιμοποιήσετε το Σύστημα Διαχείρισης

### Βήμα 1: Πρόσβαση στο Admin Panel

1. **Από το κινητό σας**, ανοίξτε τον browser (Chrome, Safari κλπ)
2. Πηγαίνετε στη διεύθυνση: **`https://yourwebsite.com/admin/`**
3. Προσθέστε αυτή τη σελίδα στα **Αγαπημένα** για γρήγορη πρόσβαση

**💡 ΣΥΜΒΟΥΛΗ:** Δημιουργήστε shortcut στην αρχική οθόνη του κινητού σας:
- **iPhone:** Safari → Κοινή Χρήση → "Προσθήκη στην Αρχική Οθόνη"
- **Android:** Chrome → Menu (⋮) → "Προσθήκη στην αρχική οθόνη"

---

### Βήμα 2: Πρώτη Σύνδεση (Μόνο 1η Φορά)

Την πρώτη φορά που θα μπείτε:

1. Θα σας ζητηθεί να **ορίσετε έναν κωδικό PIN**
2. Επιλέξτε έναν **4-6 ψήφιο κωδικό** που θα θυμάστε εύκολα
3. Επιβεβαιώστε τον κωδικό
4. ✅ **Έτοιμο!** Ο κωδικός αποθηκεύτηκε στο κινητό σας

**⚠️ ΣΗΜΑΝΤΙΚΟ:**
- Κρατήστε τον κωδικό σε ασφαλές μέρος
- Αν ξεχάσετε τον κωδικό, επικοινωνήστε με τον developer

---

### Βήμα 3: Καθημερινή Χρήση

#### 🔑 Είσοδος
1. Πηγαίνετε στο admin URL
2. Εισάγετε τον PIN σας
3. ✅ Επιλέξτε "Απομνημόνευση για 30 ημέρες" για να μην χρειάζεται PIN κάθε φορά
4. Πατήστε **"Είσοδος"**

---

## 📸 Πώς να Προσθέσετε Φωτογραφίες

### Μέθοδος 1: Από το Κινητό (Εύκολη!)

1. **Πατήστε** το μεγάλο ροζ κουμπί **"Προσθήκη Φωτογραφίας"**

2. **Επιλέξτε Φωτογραφίες:**
   - Πατήστε στην περιοχή ανεβάσματος
   - Επιλέξτε μία ή περισσότερες φωτογραφίες από το κινητό σας
   - Θα δείτε προεπισκόπηση των φωτογραφιών

3. **Επιλέξτε Κατηγορία:**
   - Γάμοι
   - Βαπτίσεις
   - Εκδηλώσεις
   - Ανθοδέσμες

4. **Προσθέστε Περιγραφή (Προαιρετικό):**
   - Λεζάντα στα Ελληνικά (π.χ. "Νυφική Ανθοδέσμη")
   - Caption στα Αγγλικά (π.χ. "Bridal Bouquet")

5. **Πατήστε "Ανέβασμα"**
   - Θα δείτε την πρόοδο της μεταφόρτωσης
   - ✅ Όταν ολοκληρωθεί, οι φωτογραφίες εμφανίζονται αυτόματα στην ιστοσελίδα!

---

## 🖼️ Διαχείριση Φωτογραφιών

### Προβολή Φωτογραφιών

- Οι φωτογραφίες εμφανίζονται σε πλέγμα
- Χρησιμοποιήστε τα **φίλτρα** (Όλα, Γάμοι, Βαπτίσεις κλπ) για να δείτε συγκεκριμένες κατηγορίες

### Επεξεργασία/Διαγραφή

1. **Πατήστε** πάνω σε μία φωτογραφία
2. Θα δείτε τις λεπτομέρειες της
3. **Επιλογές:**
   - **"Απόκρυψη"**: Η φωτογραφία κρύβεται από την ιστοσελίδα (χωρίς να διαγραφεί)
   - **"Εμφάνιση"**: Επαναφέρει κρυμμένη φωτογραφία
   - **"Διαγραφή"**: Διαγράφει οριστικά τη φωτογραφία (με επιβεβαίωση)

## ❓ Συχνές Ερωτήσεις

### 1. "Πόσες φωτογραφίες μπορώ να ανεβάσω;"
- Μέχρι **πολλές φωτογραφίες ταυτόχρονα**
- Κάθε φωτογραφία μέχρι **5MB**
- Το σύστημα συμπιέζει αυτόματα τις φωτογραφίες για γρήγορη φόρτωση

### 2. "Οι φωτογραφίες εμφανίζονται αμέσως στην ιστοσελίδα;"
- **ΝΑΙ!** Μόλις ανεβάσετε μία φωτογραφία, εμφανίζεται αυτόματα
- Οι επισκέπτες θα τη δουν αμέσως

### 3. "Μπορώ να αλλάξω τη σειρά των φωτογραφιών;"
- Οι νέες φωτογραφίες εμφανίζονται πρώτες
- Οι παλαιότερες εμφανίζονται μετά

### 4. "Τι γίνεται αν ξεχάσω τον PIN;"
- Επικοινωνήστε με τον developer για επαναφορά

### 5. "Μπορεί κάποιος άλλος να δει το admin;"
- **ΟΧΙ!** Το admin δεν εμφανίζεται πουθενά στην ιστοσελίδα
- Μόνο εσείς γνωρίζετε το URL και τον PIN
- Οι μηχανές αναζήτησης (Google) δεν το βλέπουν

### 6. "Χρειάζεται Internet για να δουλέψει;"
- **ΝΑΙ** - Χρειάζεστε σύνδεση στο Internet
- Δουλεύει με WiFi ή δεδομένα κινητής
